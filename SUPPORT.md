# Support

## Public Repository Scope

This repository is a public Windows-first handoff package for Stock Vault.

That means support requests should preferably focus on:

- public documentation
- repository structure
- Windows-facing public materials
- future macOS planning requests

## How To Ask For Help

### Documentation or repository issues

Please open a GitHub issue and choose the template that fits best.

### macOS requests

If you want future macOS support or related materials, please use the dedicated macOS request issue template.

### Security concerns

Please follow the process in [SECURITY.md](./SECURITY.md) instead of posting sensitive details publicly.

## What To Include

When asking for help, try to include:

- a short summary
- what you expected
- what happened instead
- relevant screenshots or links
- whether your request is Windows-related, documentation-related, or about future macOS support

## Response Notes

Because this is a public handoff repository, support may focus first on:

- clarifying scope
- improving documentation
- keeping Windows-first messaging accurate
- collecting future macOS demand in a structured way
