# Stock Vault Public Release v1.0.0

台灣股市專屬的本地桌面記帳工具。直接串接 TWSE / TPEX、支援 TDCC 對帳單匯入、FIFO 精確配對。資料存在自己電腦，不上雲。Windows 綠色版免安裝。

Stock Vault is a Taiwan-stock-market-focused local desktop bookkeeping tool with direct TWSE / TPEX connectivity, TDCC statement import support, precise FIFO matching, local-only data storage, and a no-install Windows portable edition.

目前這份公開 handoff 對應的 App 版本為 `v1.0.0`。

This public handoff currently maps to app version `v1.0.0`.

## 這次發佈的重點 / Release Highlights

這次是 `股市亨通` 首次以公開 GitHub 形式整理對外發佈。

This is the first public GitHub-facing release for `Stock Vault`.

這次發佈的目標，不是把它包裝成已完成的跨平台正式商業版，而是先把一個清楚、可分享、可協作、可持續維護的 Windows-first 公開入口建立起來。

The goal of this release is not to present a finished cross-platform commercial launch. It is to establish a clear, shareable, collaboration-ready, and maintainable Windows-first public entry point.

## 核心特色 / Core Product Value

- 台灣股市專屬 / built specifically for the Taiwan stock market
- 直接串接 TWSE / TPEX / direct TWSE / TPEX connectivity
- 支援 TDCC 對帳單匯入 / TDCC statement import support
- FIFO 精確配對 / precise FIFO matching
- 資料保留在自己的電腦，不上雲 / local-only data storage, not cloud-first
- Windows 綠色版免安裝 / no-install Windows portable workflow
