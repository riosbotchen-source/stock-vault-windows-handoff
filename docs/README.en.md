# Stock Vault

This repository is a public GitHub handoff package for Stock Vault, positioned as a Windows-first open-source presentation repo.

[Back to Home](../README.md) | [中文版](./README.zh-TW.md)

## Project Positioning

This repository is intended to:

- present the project publicly
- provide Windows-related handoff materials
- serve as an open-source sharing entry point
- leave room for future macOS expansion

## Current Status

| Item | Status |
| --- | --- |
| Windows version | Currently available |
| macOS version | Not included in this public package |
| Future macOS support | Can be provided later if needed |
| Open-source status | Public |
| License | MIT |

## Current Public Contents

This repository currently contains a shareable public handoff package rather than the complete cross-platform source tree.

It currently includes:

- public-facing project documentation
- the MIT license
- selected test files
- representative Tauri / Rust core files

## Platform Notes

### Windows

- the current public reference platform
- the main focus of this repository
- the preferred platform for presentation and handoff at this stage

### macOS

- not included in this public package yet
- can be prepared and provided later if needed
- should ideally be accompanied by build, signing, and platform-difference documentation

Reserved platform structure:

- [Platform Guide](./platforms/README.md)
- [Windows Page](./platforms/windows.md)
- [macOS Page](./platforms/macos.md)

## Collaboration Notes

If macOS support is added later, it is recommended to also prepare:

- macOS build instructions
- Tauri / Xcode configuration details
- platform difference notes
- release and signing workflow documentation

## Important Note

This public repository is intentionally scoped to show the currently shareable materials in a clear and honest way.

That is why it explicitly states:

- Windows is the current primary platform
- macOS materials can be provided later
- this repository is a public handoff package
