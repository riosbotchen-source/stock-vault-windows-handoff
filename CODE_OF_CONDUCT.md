# Code of Conduct

## Our Standard

We want this repository to feel constructive, respectful, and safe for collaboration.

Examples of expected behavior:

- being respectful in discussion
- giving clear and helpful feedback
- focusing on the work instead of the person
- welcoming contributors with different backgrounds and experience levels
- making room for correction without hostility

Examples of unacceptable behavior:

- harassment or personal attacks
- insulting or demeaning language
- deliberate intimidation
- publishing private information without permission
- bad-faith disruption of discussion or review

## Enforcement

Repository maintainers may remove, edit, or reject comments, issues, pull requests, and other contributions that do not align with this Code of Conduct.

## Reporting

If you experience or witness unacceptable behavior, please contact the maintainer privately when possible.

## Project Tone

This repository is a public handoff package. Clear, calm, and honest communication matters a lot here, especially when discussing platform support, repository scope, and future macOS plans.
