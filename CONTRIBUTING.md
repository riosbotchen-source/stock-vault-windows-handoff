# Contributing to Stock Vault

Thanks for your interest in contributing.

This repository is a public Windows-first handoff package for Stock Vault. It is intentionally narrower than a full internal product repository, so the best contributions are the ones that improve clarity, maintainability, and future collaboration.

## Best Contribution Types

- documentation improvements
- README polish
- translation fixes
- issue clarification
- repository organization
- Windows-related handoff improvements
- future macOS planning notes

## Before You Open a Pull Request

Please make sure your change:

- fits the public handoff scope of this repository
- does not imply unsupported platform availability
- does not add confidential or private materials
- keeps the current Windows-first positioning accurate

## Platform Positioning

At the moment:

- Windows is the primary public platform
- macOS is not included yet
- macOS materials may be added later if needed

Please keep that messaging consistent in documentation and pull requests.

## Pull Request Guidelines

- keep changes focused
- explain what changed and why
- prefer small, reviewable pull requests
- update related documentation when needed
- avoid bundling unrelated changes together

## Issues

When opening an issue, please include:

- a clear summary
- expected outcome
- current outcome
- any relevant screenshots or file references
- whether the report is Windows-specific or a future macOS request

## Security

If you believe you found a security issue, please do not open a public issue first. See [SECURITY.md](./SECURITY.md).
